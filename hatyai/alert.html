📣 รับแจ้งเตือนระดับน้ำของอำเภอคุณ

เมื่อสถานีน้ำใกล้บ้านคุณระดับสูงถึงเกณฑ์เฝ้าระวัง ระบบจะส่งข้อความเข้า LINE ให้ทันที — ไม่ต้องเปิดเว็บเฝ้าเอง เหมาะเวลาน้ำมาตอนกลางคืน

1
เพิ่มเพื่อน LINE Official Account ของระบบ หรือเข้ากลุ่ม LINE ของอำเภอคุณ (ลิงก์/QR ด้านล่าง)
2
กด เพิ่มเพื่อน — เท่านี้ก็จะได้รับข้อความเตือนอัตโนมัติ
3
ต้องการหยุด บล็อก OA หรือออกจากกลุ่มได้ทุกเมื่อ
💚 เพิ่มเพื่อน LINE OA ยังไม่ได้ตั้งค่าลิงก์ — สมาชิกตั้งได้ในส่วนจัดการด้านล่าง

หมายเหตุ: LINE Notify ปิดบริการแล้ว (มี.ค. 2025) ระบบจึงใช้ LINE Messaging API (LINE OA) ในการส่งแจ้งเตือน
