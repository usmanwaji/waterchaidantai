# telerid-scraper — ดึงภาพกล้อง+ระดับน้ำ โทรมาตร ชป. (5 จังหวัดใต้)

ดึงภาพกล้อง CCTV + ระดับน้ำ ของสถานีโทรมาตร [telerid.rid.go.th](https://telerid.rid.go.th/)
ใน **สตูล · สงขลา · ปัตตานี · ยะลา · นราธิวาส** แล้วส่งภาพขึ้นสาขา `cam` ให้ dashboard แสดง

> ⚠️ **ต้องรันบนคอมที่ IP ไทย** — telerid บล็อก IP เซิร์ฟเวอร์ต่างประเทศ (GitHub Actions/Cloudflare ใช้ไม่ได้)
> จึงต้องรันบนคอมของท่านเอง (ที่ออฟฟิศ/ที่บ้าน ซึ่งเป็นเน็ตไทย)

---

## ตั้งค่าครั้งแรก (ทำครั้งเดียว ~10 นาที)

### 1) ลง Node.js
- โหลดจาก https://nodejs.org (เลือกปุ่ม **LTS**) → ติดตั้งแบบกด Next ไปเรื่อย ๆ จนเสร็จ

### 2) สร้าง "กุญแจ" ให้สคริปต์ส่งภาพขึ้น GitHub ได้ (Personal Access Token)
1. เข้า https://github.com/settings/tokens?type=beta → กด **Generate new token**
2. ตั้งชื่ออะไรก็ได้ เช่น `telerid`
3. **Repository access** → เลือก **Only select repositories** → เลือก `waterchaidantai`
4. **Permissions → Repository permissions → Contents** → เลือก **Read and write**
5. กด **Generate token** → **คัดลอกโทเคน** (ขึ้นต้น `github_pat_...`) เก็บไว้ (โชว์ครั้งเดียว)

### 3) บอกคอมให้จำโทเคน (ทำครั้งเดียว)
- เปิด **Command Prompt** (พิมพ์ `cmd` ที่ช่องค้นหา Windows)
- พิมพ์บรรทัดนี้ (แทนที่ `วางโทเคนตรงนี้` ด้วยโทเคนจากข้อ 2):
  ```
  setx GH_TOKEN "วางโทเคนตรงนี้"
  ```
- กด Enter → **ปิดหน้าต่าง Command Prompt** (สำคัญ: ต้องปิด-เปิดใหม่ค่าถึงจะมีผล)

## วิธีใช้งาน (ทุกครั้ง)

ไปที่โฟลเดอร์ `telerid-scraper` แล้ว **ดับเบิลคลิกไฟล์ `run-local.bat`**

- ครั้งแรกจะติดตั้งของที่จำเป็นก่อน (รอ 2-3 นาที) ครั้งต่อไปจะเร็ว
- มันจะ: เปิด telerid → ดึงภาพกล้องทุกสถานี → ส่งขึ้น GitHub สาขา `cam`
- เสร็จแล้วเปิดเว็บหน้า **สถานการณ์** → เปิดชั้น **โทรมาตร ชป.** → คลิกหมุด ▲ เห็นภาพเลย

---

## ให้มันดึงเองอัตโนมัติ (ไม่บังคับ)

ใช้ **Task Scheduler** ของ Windows ตั้งให้เปิด `run-local.bat` ทุก 15-30 นาที
(คอมต้องเปิดอยู่ตอนมันรัน) — บอกผมได้ถ้าอยากให้แนะนำวิธีตั้งละเอียด

---

## ถ้าดึงภาพได้ 0 ตัว

เปิดไฟล์ `telerid-cam/stations.json` ดูค่า `withImage` และ `tokenCaptured`
พร้อมไฟล์ `telerid-cam/_discovery.json` → ส่ง 2 ไฟล์นี้ให้ผมปรับสคริปต์

**ถ้าจำเป็นต้องล็อกอิน RID** (anonymous ไม่ได้จริง ๆ): เปิด Command Prompt ตั้งค่าเพิ่ม
```
setx RID_USER "ชื่อผู้ใช้ RID"
setx RID_PASS "รหัสผ่าน RID"
```
แล้วปิด-เปิดใหม่ ค่อยดับเบิลคลิก `run-local.bat` อีกครั้ง
(รหัสอยู่ในเครื่องท่านเท่านั้น ไม่ถูกส่งขึ้นเว็บ)
