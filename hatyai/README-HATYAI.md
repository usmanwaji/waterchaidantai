# One Map หาดใหญ่ — บันทึกการปรับขอบเขต (15 ก.ค. 2026)

เดิม: แดชบอร์ด 5 จังหวัดชายแดนใต้ (สตูล สงขลา ปัตตานี ยะลา นราธิวาส)
ใหม่: **อ.หาดใหญ่ จ.สงขลา + อำเภอต้นน้ำลุ่มน้ำคลองอู่ตะเภา** (ใช้งานในเครื่องเท่านั้น ไม่ deploy ขึ้น GitHub)

## ขอบเขตข้อมูล
ยึดหลักการเตือนภัยล่วงหน้า: น้ำท่วมหาดใหญ่มาจากลุ่มน้ำคลองอู่ตะเภา ฝน/ระดับน้ำที่อำเภอต้นน้ำให้เวลาเตือนล่วงหน้า 6–12 ชม. จึงเก็บ 6 อำเภอ: **หาดใหญ่ · สะเดา · คลองหอยโข่ง · นาหม่อม · บางกล่ำ · ควนเนียง** (ชุด `BASIN_AMPHOE` ในโค้ด — แก้ได้ทุกไฟล์ที่ประกาศ)

## สถานีหลัก (หน้าแรก) — ลำดับต้นน้ำ→ในเมือง
- **X.173A** บ้านม่วงก็อง อ.สะเดา (ตลิ่ง +16.13 ม.รทก.)
- **X.90** บ้านบางศาลา ต.บ้านพรุ (ตลิ่ง +9.34)
- **X.44** บ้านหาดใหญ่ใน ในเมือง (ตลิ่ง +7.15 · วิกฤต +7.44)
ค่าตลิ่งดึงจาก API สด (min_bank) อัตโนมัติ ค่าในไฟล์เป็นค่าสำรอง · เส้นเฝ้าระวัง (warn) ตั้งไว้ประมาณการ ควรปรับตามเกณฑ์ ปภ./ชป. จริง

## สิ่งที่เปลี่ยนหลัก ๆ
- ทุกหน้า: จำกัด province_code=90 และกรองสถานีเฉพาะอำเภอลุ่มน้ำ ปุ่มกรองเปลี่ยนจากรายจังหวัดเป็นรายอำเภอ แผนที่เริ่มที่หาดใหญ่
- `js/ddpm_risk_data.js`: จุดเสี่ยง ปภ. กรองเหลือ 535 จุด (จาก 3,434)
- `js/rivers.js`: วาดเฉพาะคลองอู่ตะเภา คลองสาขา และคลองระบายน้ำ ร.1
- `js/shelters_nwt.js` + ข้อมูลตั้งต้นใน `shelter.html`: ลบชุดนราธิวาสออก — ศูนย์พักพิงหาดใหญ่เพิ่มผ่านฟอร์ม/Supabase หรือใส่ในไฟล์นี้
- `forecast.html`: การ์ดเขื่อนเปลี่ยนจากบางลางเป็นอ่างฯ คลองสะเดา (ถ้า API ไม่มีข้อมูล การ์ดจะไม่แสดง)
- `history.html`/`repeat.html`: สถิติ GISTDA นับได้ระดับจังหวัด (ข้อจำกัด API) — แผนที่ซูมลุ่มน้ำ
- GitHub: ลบ `.github/` (workflow) และ git remote ออกแล้ว — โปรเจกต์อยู่ในเครื่องอย่างเดียว
- PWA: bump cache เป็น `oms-hatyai-v1` กันหน้าเก่า 5 จังหวัดค้างแคช

## วิธีใช้ในเครื่อง
เปิด `index.html` ตรง ๆ หรือรันเซิร์ฟเวอร์ท้องถิ่น เช่น `python -m http.server 8000` แล้วเข้า http://localhost:8000
